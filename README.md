# Electronics-Repair-Business-Website
A modern, responsive, and SEO-friendly business website built for S.K. Electronics &amp; Repairing Centre, an electronics repair service based in Lalitpur, Nepal. The website is designed to provide customers with an easy way to explore services, learn about the business, and book repair appointments through a clean, fast, and user-friendly interface.
# S.K. Electronics & Repairing Centre

A modern, responsive electronics repair service website built with Next.js 15, React, TypeScript, Tailwind CSS, and Framer Motion.
